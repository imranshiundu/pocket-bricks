# Gameplay Notes

Pocket Bricks uses a classic 10 x 20 board and seven block shapes. The design target is not a modern competitive block game. The target is a fast, readable, old mobile-phone feeling that works comfortably on smartphone touch screens.

## Rules

- Move blocks left, right, or down.
- Rotate clockwise.
- Clear full horizontal lines.
- The game ends when a new block cannot spawn.
- Level increases every 10 cleared lines.
- Falling speed increases as the level rises.

## Scoring

| Lines cleared | Base score |
| --- | ---: |
| 1 | 40 x level |
| 2 | 100 x level |
| 3 | 300 x level |
| 4 | 1200 x level |

Hard drop adds 2 points per dropped cell.

## Feel rules

- Keep the game monochrome.
- Keep the board readable under thumb play.
- Keep restarts instant.
- Do not add heavy menus or distracting animations.
