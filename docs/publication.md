# Publication Checklist

Use this checklist before sharing the repository.

- Repository is public.
- README shows the screenshot.
- License is present.
- CI is present.
- GitHub Pages workflow is present.
- No proprietary phone/game logos are included.
- No copied sounds or ROM assets are included.
- The app runs as static files.
- Tests pass with `npm test`.

## GitHub Pages

The included workflow deploys the repository contents as a static site. If Pages is not active, enable GitHub Pages in the repository settings and choose GitHub Actions as the source.
