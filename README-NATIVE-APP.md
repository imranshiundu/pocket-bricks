# Pocket Bricks native app notes

Pocket Bricks is available as:

- a Vercel-hosted web game for quick access, and
- a real Android APK built from `native-android/`.

The APK is the correct option for users who want a proper installed Android app. It is not a PWA shortcut.

## Download the APK

Go to GitHub Releases and download `Pocket-Bricks.apk`:

https://github.com/imranshiundu/pocket-bricks/releases

## Build it

Use the workflow called **Build Native Android APK** or build locally from `native-android/`.

## Termux

```bash
bash scripts/install-android-termux.sh
```
