# Contributing

Pocket Bricks is intentionally small. Contributions should protect the old-phone feel and keep the app fast on low-end smartphones.

## Good contributions

- Gameplay bug fixes.
- Accessibility improvements.
- Better mobile input handling.
- Small performance improvements.
- Tests for game logic.
- Documentation improvements.

## Avoid

- Ads, analytics, accounts, payments, or tracking.
- Heavy frameworks for the core app.
- Proprietary assets or copied sounds.
- Brand names, logos, or assets owned by phone/game companies.
- Complicated skins or modern arcade features that erase the simple feel.

## Local workflow

```bash
npm install
npm test
npm run serve
```

Open `http://localhost:4173` and test on a real phone when changing controls or layout.
