# Security Policy

Pocket Bricks is a static browser game. It does not require accounts, servers, databases, analytics, or user data.

## Supported version

Only the `main` branch is maintained.

## Reporting

Open a GitHub issue for security-sensitive bugs that do not expose private information. If a bug involves a real exploit, do not include working exploit details publicly. Contact the maintainer first.

## Data handling

The app stores only a local high score and sound preference in the user's browser local storage.
